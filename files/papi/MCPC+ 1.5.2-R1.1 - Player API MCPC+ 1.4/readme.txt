================
Player API MCPC+
================

Version 1.4 for MCPC+ 1.5.2-R1.1

by Divisor



Description
===========

Player API MCPC+ is an API which gives mods access to the class and "EntityPlayerMP" aka "jc" while minimizing conflicts between mods.



Installation
============

Copy all class files inside this ZIP file to their corresponding locations in your "mcpc-plus-1.5.2-R1.1.jar".

In any case, NEVER forget: ALWAYS back up your stuff!
