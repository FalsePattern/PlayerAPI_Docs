================
Player API MCPC+
================

Version 1.0 for MCPC+ 1.6.4-R0.1

by Divisor



Description
===========

Player API MCPC+ is an API which gives mods access to the class and "EntityPlayerMP" aka "jv" while minimizing conflicts between mods.



Installation
============

Copy all class files inside this ZIP file to their corresponding locations in your "mcpc-plus-1.6.4-R0.1.jar".

In any case, NEVER forget: ALWAYS back up your stuff!
